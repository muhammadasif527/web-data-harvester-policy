Privacy Policy for Web Data Harvester

Effective Date: May 2026

Web Data Harvester (“the extension”, “we”, “our”, or “us”) respects your privacy and is committed to protecting user data. This Privacy Policy explains how the extension handles information when you use our Chrome extension.

Information We Collect

Web Data Harvester is designed to extract website data based on user actions. The extension may temporarily process the following types of information:

Website content selected by the user for extraction
Images, links, tables, text, and structured webpage data
User-defined scraping settings and export preferences
Authentication/session data required to access webpages already open in the user’s browser session

The extension does not collect personal information unless it is part of the content the user intentionally chooses to extract from a webpage.

How We Use Information

The collected data is used solely to provide the extension’s functionality, including:

Extracting website data selected by the user
Exporting data into CSV, JSON, Excel, or similar formats
Downloading images, links, and files
Supporting pagination, deep crawling, and automated extraction workflows
Saving user preferences and scraping configurations locally
Data Storage

Web Data Harvester primarily stores settings and temporary extraction data locally within the user’s browser using Chrome storage APIs.

We do not sell, rent, or share user data with third parties.

Permissions Usage

The extension requests certain Chrome permissions to enable its core functionality:

activeTab — Access the currently active webpage after user interaction
tabs — Manage scraping workflows across browser tabs
scripting — Inject extraction scripts into webpages
storage — Save user settings and scraping configurations
downloads — Export and save extracted files
cookies — Access authenticated webpage sessions for user-requested scraping
webRequest — Detect and process network requests for dynamic data extraction
notifications — Notify users about completed tasks and exports
alarms — Schedule automated scraping and crawling tasks
Host permissions — Access websites selected by the user for scraping
Third-Party Services

Web Data Harvester does not sell collected data to advertisers or third parties. If optional cloud or AI features are added in the future, this policy will be updated accordingly.

Security

We take reasonable measures to protect user data and minimize unnecessary data access. Data processing is limited to functionality explicitly initiated by the user.

User Control

Users remain fully in control of the extension and the data they choose to extract. Users can remove the extension at any time through Chrome browser settings.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected by updating the effective date above.
